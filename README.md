# JavaScript Form Validation

## Overview

This project is a responsive job application website developed using HTML, CSS, and JavaScript. It includes a professional landing page, application form, and client-side validation for user input.

## Features

* Landing page with Apply Now button
* Job application form
* Name validation
* Email validation
* Phone number validation
* Message validation
* Responsive design
* User-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Learning Outcomes

* Learned DOM manipulation using JavaScript
* Implemented form validation techniques
* Improved user experience with error handling
* Built responsive web forms
